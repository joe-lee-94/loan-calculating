이 계산기와의 연결

계산기에서 가구 유형을 선택하면, 해당 유형의 소득 상한이 자동으로 적용됩니다.
입력한 부부 합산 연소득이 기준을 초과하면 자격 판정에서 바로 표시됩니다.

✅일반 가구 선택 → 7,000만원 이하 기준 자동 적용
✅신혼가구 선택 → 8,500만원 이하 기준으로 판정
✅다자녀 선택 → 1억원 이하 + 대출한도 4억원 확대
⚠️계산기 입력값은 빠른 확인용이며, 실제 심사와 다를 수 있습니다
